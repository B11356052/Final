## 📁 資料集 1

| 項目 | 內容 |
|------|------|
| 資料集名稱 | Netflix Movies and TV Shows |
| 來源平台 | Kaggle |
| 資料格式 | CSV |
| 資料筆數（大約） | 約 8,000 筆 |
| 主要欄位 | title、director、cast、country、release_year、rating、duration、listed_in |
| 可以解決的問題 | 可以分析 Netflix 上不同類型影片的分布，例如哪些類型最熱門、不同年份上架數量變化，以及各國影視作品比例等 |
| 資料品質（五大原則） | 完整性：部分欄位有缺失（如 director）  
一致性：資料格式統一  
正確性：資料來源可信  
時效性：可能不是最新資料  
唯一性：資料重複情況少 |

## 📁 資料集 2

| 項目 | 內容 |
|------|------|
| 資料集名稱 | 台灣空氣品質指標（AQI） |
| 來源平台 | data.gov.tw |
| 資料格式 | JSON / CSV |
| 資料筆數（大約） | 數千筆（持續更新） |
| 主要欄位 | SiteName、County、AQI、PM2.5、PM10、O3、PublishTime |
| 可以解決的問題 | 可以用來分析台灣各地空氣污染狀況，例如哪個縣市空氣品質較差、PM2.5變化趨勢，以及污染較嚴重的時間點 |
| 資料品質（五大原則） | 完整性：資料欄位齊全  
一致性：資料格式統一  
正確性：政府資料可信度高  
時效性：資料持續更新，時效性高  
唯一性：以測站為單位，無明顯重複資料 |

## 📌 結論

本次選擇的兩份資料集分別屬於不同領域，一個是娛樂產業（Netflix），另一個是環境監測（AQI）。  

透過這些資料，可以應用在不同的分析方向，例如：

- 商業分析（影片推薦與市場趨勢）
- 環境分析（空氣污染與健康影響）

透過比較不同資料集，也能更了解資料品質對分析結果的重要性。
